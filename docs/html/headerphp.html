Extraction of page first name

Extraction of page first name contains the following code.


<?php /* php to extract page name*/
$directoryURI = $_SERVER['REQUEST_URI'];
$path = parse_url($directoryURI, PHP_URL_PATH);
$components = explode('/', $path);
$first_part = $components[2];
$first_part = basename($_SERVER['PHP_SELF'], ".php");
include('includes/db.php');
?>


$_SERVER['REQUEST_URI'] requests the url of the current page & stores in $directoryURI variable.

parse_url($directoryURI, PHP_URL_PATH) parses the url into php readable url & stores in $path variable.

$components is an array that uses explode function to parse $path variable & stores each text block into each array space.

$first_part stores the third component of the components array i.e. page name.

basename($_SERVER['PHP_SELF'], ".php") strips off .php from the first name & page name is obtained in $first_part.

<head> components

Meta elements tag provides metadata about the HTML document. Metadata are typically used to specify page description, keywords, author of the document, last modified, and others.

<meta charset="utf-8"> sets to utf-8. utf-8 is character library.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> is for responsive web design. It is a viewport for meta elements.


Link defines a link between a document and an external resource.


<link rel="stylesheet" type="text/css" href="/bistro/css/main.css"><!-- Main CSS -->
<link rel="stylesheet" type="text/css" href="/bistro/css/bootstrap.min.css"><!-- Bootstrap CSS --> 
<link rel="shortcut icon" type="image/png" href="/bistro/images/logo/favicon.png"/><!-- Link for favicon --> 


The first one is main.css. The second one is link for bootstrap.min.css. The third is the shortcut icon or the favicon is the small png icon that appears beside the Title bar.


Hover over the picture to see the details.


Script  is used to define a client-side script (JavaScript).

The <script> element either contains scripting statements, or it points to an external script file through the src attribute.


<!-- Scripts -->
 <script src="/bistro/icons/friconix.js"></script><!-- Icons from friconix.com -->
 <script src="/bistro/js/jquery.js"></script><!-- jQuery JS -->
 <script src="/bistro/js/bootstrap.min.js"></script><!-- Bootstrap JS -->



The first one is friconix.js, an icons provider. The second one is link for jQuery.js, a broad library for easy use of javascript in the website.


The third is for bootstrap.min.js, bootstrap.min.js is the js pack for Bootstrap used along with bootstrap.min.css & increases the interactability of the Bootstrap components.

<body> components

<body> consists of the <header>, which defines all the head parts of the website. Head part includes logo and the navigation bar.


Under <nav>,


<nav class="navbar navbar-expand-md navbar-light"><!-- Navigation Bar -->


Our navbar is a Bootstrap navbar. Bootstrap classes navbar, navbar-expand-md & navbar-light defines the styles of the navbar.


navbar specifies that it is a Bootstrap navbar.

navbar-expand-md specifies navbar should collapse into a "Hamburger menu" while screen size is less than 'md' or medium size, i.e. less than 768px. In contrast, expanding of the navbar when the screen sizes increases than 768 px can also be said.
navbar-light specifies it's a light coloured navbar.


All of the above are Bootstrap classes. Learn them at https://getbootstrap.com/docs/

Hamburger Icon

This shows up when the menu is not adjustable in screen, when screen size becomes less than 768px. It is a part of responsive design. This is also generated by Bootstrap.


Hover over the picture to see the details.


<button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#nav">
    <span class="navbar-toggler-icon"></span>
</button><!-- Hamburger icon (visible in <768px devices) -->


It is a button having class navbar-toggler, data-toggle="collapse" and data-target="#nav".


navbar-toggler gives it it's style. It is a bootstrap class.

data-toggle="collapse" is attribute used by bootstrap.min.js to specify that it is a collapsible entity.

data-target="#nav" specifies that it is used to toggle the items of #nav id entities.


Finally, <span class="navbar-toggler-icon"></span> is what places the three dashed icon or popularly known as "hamburger icon".

Navbar

The visible navbar is contained within the section. The heading part code of navbar is shown below:


<div class="collapse navbar-collapse" id="nav"><!-- Navbar, the collapsible part -->
    <ul class="navbar-nav ml-auto"><!-- Navbar start -->


In <div class="collapse navbar-collapse" id="nav">, collapse specifies that it should be given collapse Bootstrap class. It has a id of nav. This id is what through which Bootstrap makes this section collapsible in small screens. Both of the classes are used as styling entities.


Bootstrap uses ul as the parent element for navbar. li are corresponding list items or navigation bar items list item. In, <ul class="navbar-nav ml-auto">, navbar-nav  specifies that ul holds the nav & ml-auto adds a left margin to it. Thus, pushing it to the right.
